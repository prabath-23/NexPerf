# NexPerf v0.2.0

v0.2.0 establishes NexPerf as a local observability service:

- Vue is the primary dashboard.
- Go serves APIs and production Vue assets.
- SQLite stores historical CPU, memory, and disk samples.
- The monitoring loop samples every 5 seconds.
- `start`, `stop`, and `open` manage the local service lifecycle.
- The frontend is split into reusable observability components.
